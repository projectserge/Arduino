# Basics for playing with Arduino.
First you'll need an Arduino.

![](https://upload.wikimedia.org/wikipedia/commons/3/38/Arduino_Uno_-_R3.jpg)

Then you'll need a breadboard.

![](https://upload.wikimedia.org/wikipedia/commons/thumb/7/73/400_points_breadboard.jpg/800px-400_points_breadboard.jpg)

You'll need some jumper wires.

![](https://upload.wikimedia.org/wikipedia/commons/5/5c/A_few_Jumper_Wires.jpg)

You'll need a USB cable to connect your Arduino to your computer.

![](https://upload.wikimedia.org/wikipedia/commons/6/62/USB_Type-B_plug_coloured.svg)

The rest is all about finding the right sensors and parts to use in your creations.

Also, I'm new to the world of Github so please bear with me as I hash out all the details.
My hope is to post all of my latest projects and code here on Github for you to use.

As always if you find a better way, by all means, please share the knowledge.
